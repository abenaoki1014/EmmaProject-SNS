概要：
cssについて
    CSSとは、HTMLで書かれたページをWebブラウザやメーラーにどのように表示するかを定義するための言語です。

実際に使う記述方式：
    /* 大見出しの設定 */　コメントです
    color: #ff0000　カラー指定です
    font-size: 24px　文字サイズ指定です
    font-weight: 800　文字の太さ指定です
    font-family:Helvetica,Arial,"ヒラギノ角ゴ Pro W3",
    "Hiragino Kaku Gothic Pro","ＭＳ Ｐゴシック","MS PGothic",sans-serif;　文字のフォント指定です。
    text-align: left;   文字の行そろえです。
    text-decoration: none;　文字の装飾方法を指定しています。
    line-height: 36px;　文字の行の高さを指定します。（行間）
    background-color: #000000;　背景職を指定します
    background-image: url("https://sapana.co.jp/img/sample.png");　画像背景を設定します
    border: 1px solid #333333;　上下左右のボーダーラインの太さ、スタイル、色をまとめて指定します。
    margin: 25px; /* 上下左右の一括指定 */　上下左右の外側余白設定
    padding: 10px; /* 上下左右の一括指定 */ 上下左右の内側余白
    width: 100%;　ブロック要素の横幅指定
    height: 180px;　ブロック要素の立幅指定

基本的なCSSセレクタ：
    CSSセレクタとは、どの部分にCSSを適用するかを指定するパスのようなものです。
    CSSを使いこなすためには、意図した通りのセレクタ指定ができることがとても重要です。
    タグ名のセレクタ指定：
        html、body、h1、p、a、img、table、ul、li、など、HTMLのタグ名を直接セレクタに指定する方法です。CSSにはタグ名をそのまま記述します。
        先頭にシャープ（#）やドット（.）は付けません。

    #id名によるセレクタ指定：
        HTMLソース内のid名をセレクタに指定する方法です。CSSのid名の先頭にはシャープ（#）を付けて記述します。
        



CSSファイルの読み込み方：
    これには、三つあります。
    インライン形式  HTMLのタグの中に、直接CSSのスタイルを記述していく方法です。
    ヘッダー埋め込み形式　head要素内に、style要素を使って一括してCSSを記述する方法です。
    外部参照　SNSアプリで使用しているのは、これです。
